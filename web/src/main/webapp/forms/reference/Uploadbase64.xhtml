Upload control which uses base64 for content encoding

This form control enables the common feature found on Web sites to upload a file from the local file system, as well as accepting input from various devices including microphones, pens, and digital cameras.

An upload element bound to a node with datatype base64Binary will be represented by a file upload control. The selected file will be uploaded and written to the bound node in base64-encoded format.
