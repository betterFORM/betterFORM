Enjoy your application on a Desktop PC, Laptop, Mobile Device, Kindle, your home TV or wherever you want it. Due to the XForms User Interface is abstract betterFORM can easily target any end device, from the high end web 2.0 application to dynamic form rendered as PDF, Word or Excel file.

eForms / XForms

Building top-notch form-based applications is one of the cornerstones of our expertise. We have developed form applications for customers from around the globe in many different domains.

As you may have experienced for yourself "Forms are everywhere". Our Solutions provide a pluggable and seamless integration into your current infrastructure. If it comes to strong validation and calculation features, datatyping and a highly dynamic user interface common user interface framworks often fall short or require repetitive scripting. With XForms you get the full potential of XML Schema, XPath, XML Events and a descriptive language that is powerful to use and easy to extend.

Unlike proprietary software betterFORM XForms Toolkit is based on the W3C XForms recommendation guaranteeing a long lifetime and interoperability with existing and future W3C standards.

XML Applications

If processing large amounts of data in XML-structures is your livelihood utilizing XML-Application is a natural way to go. "XML from front to back" will give one the advantage that there is no need to shred data into tables rows, just a single elegant data format trough all parts of a software solutions. With XQuery there exists an easy to use and learn query language based on established W3C-standards like XPath. Appliance of other XML-technologies like XSLT and XML-FO will broaden the capabilities even more.

Data Management

Ever dreamed of just using your XML-Data without the hassle to use a mapping to store it inside a database? Are you using established standards which provide their own XML-schema definitions like Health Level 7 or Aeronautical Information Exchange Model? Want to create usable and clear user interfaces from schema definitions or even validate your data through these? Then betterFORM XForms is the right companion for the task at hand.

EAI / SOA / EDI

In advanced technology fields like EAI, SOA or EDI but not limited to these XML is the lingua franca of data exchange as it is easy to produce and consume. This is the common ground on which hetergeneous systems can be coupled with each other without needing to use proprietary interfaces. XML allows aggregation and exchange of information between these systems.

Service-oriented architectures typically hide complex internal logic behind a service-layer to be used as an outer facade to the system. SOAP messages often play a major role in this area and are natively supported by XForms. This makes it an ideal tool to build dynamic, validating browser front-ends that directly interact with your web services. Additional validation layers are unnecessary as your forms always send valid data.

XForms applications can load as many data instances from different sources at once, pulling them together in individual views for the user, allow to act upon the data and send them back for storage or further processing by a service. XForms and other open XML technologies together can enhance the functionality of existing systems with a mature form processing engine.

Complete XML applications can address even more demanding requirements. Data collection, harvesting, Metadata management and Messaging are just examples of the broad possibilities of XML technology.
