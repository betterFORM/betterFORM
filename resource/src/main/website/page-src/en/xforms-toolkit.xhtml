The XForms Toolkit

offers advanced eForms capabilities running in your browser.

higher productivity - lower cost

The XForms Toolkit adds an abstraction layer to your web application development and lifts it to the next productivity level.

The descriptive and generic approach avoids repetitive scripting lowers the cost for development and changes. You get running applications within days.

complete W3C XForms 1.1 processor

betterFORM XForms is a XForms engine for advanced, dynamic forms that run in every major browser without any plugins.

betterFORM XForms Toolkit runs on Windows, OS X, Linux, FreeBSD and other operating systems.

Attractive, dynamic and consistent user interfaces for your data

The betterFORM engine turns XForms into state-of-the-art Web 2.0 applications which can be consumed by every major desktop or mobile browser. With a rich set of styling options you can tailor the look&feel to exactly match the guidelines of your CI.

XForms-driven applications allow to very easily aggregate various data sources into one view and to act upon that data. A generic set of actions allows data- as well as user interface manipulation without writing a single line of script code.

flexible and easy to integrate

The XForms Toolkit easily integrates with your existing infrastructure and offers extension points for plugging custom logic. It plays well with ReST, SOAP, XMLRPC and HTTP making it an ideal partner in SOA and EAI.

The complexity of the underlying technologies is shielded away while still being flexible to future requirements and agile methodologies.

completely based on Open Source and Open Standards
The XForms Toolkit is completely free Open Source software and itself exclusively uses Open Standards for future-proof solutions. The toolkit is published under the business-friendly BSD license.
