XML from front to back

betterFORM XML Suite summarises the benefits of well-tried W3C standards like XML, XForms, XQuery, XSLT / XPath, XML Schema, HTML and others in a single open & free product.

betterFORM XML Suite is a zero-translation architecture making data mapping superfluous by utilizing XML in the frontend, business layer and backend.

Increase productivity

Experience the power of rapid application development with the betterFORM XML Suite.

The betterFORM XML Suite comes with a complete toolchain to generate most parts of an application of XML Schema or alike Schema definitions.

Templating mechanisms make it easy to split applications into modules and to reuse them.

Master your data & present them as you like

The betterFORM XML Suite lets you master structured as well as unstructured data in a readable way. Thanks to the inherent Model View Controller (MVC) architecture its easy as can be to modify & present your data as you like.

Tailor you data as you need them with XQuery and transform the results into alluring User Interfaces, PDF forms or Word documents utilizing XSL.

Compose & Execute Rich Internet Applications

The betterFORM XML Suite comes with everything you need to create & run fresh Rich Internet Application. Concentrate on developing your application and profit from the build in XForms Toolkit, the excellent eXist XML database all running on the reliable Jetty Server.

Thanks to the installer you can start right away developing your application without the need of complex installations or configurations.

Completely based on Open Source and Open Standards
The XForms Toolkit is completely free Open Source software and itself exclusively uses Open Standards for future-proof solutions. The toolkit is published under the business-friendly BSD license.
