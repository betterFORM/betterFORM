The betterFORM Team:

Joern Turner is a XML application architect with special expertise in W3C XForms. He founded the Chiba project in 2001 which was the first server-side XForms implementation which ran in standard browsers.

In the last 20 years he worked as consultant, architect and developer, building enterprise solutions with Java and XML for a broad range of industries and organisations.

Lars designs and realizes applications based on XForms / XSLT / Java and Javascript. He guides customers through the whole software life cycle - starting from requirements engineering until the software is deployed.
