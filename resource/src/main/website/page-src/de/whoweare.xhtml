Das betterFORM Projekt

Wir sind ein Team von Softwareentwicklern, dass täglich an der Verbesserung der betterFORM Produkte arbeitet und es gleichzeitig in verschiedensten Projekten professionel einsetzt. Unsere Philosophie ist Offenheit: Open Source, offene, Hersteller-unabhängige Standards, offene Kommunikation mit unseren Kunden und Anwendern führen zu langlebigen, erweiterbaren Lösungen.

The betterFORM Team:

Joern Turner ist XML application architect mit spezieller Expertise im W3C XForms standard. Er gründete das Chiba Projekt, die 1. server-seitige XForms Implementierung, die in Standard-Browsern läuft bereits im Jahr 2001.

Während der letzten 20 Jahre arbeitete er als Berater, Software Architekt, Projektleiter und Entwickler und entwickelte Enterprise Anwendungen mit Java und XML-Technologien für eine breite Palette von Anwendungsbereichen.

Lars Windauer ist ein XML Experte mit dem Hauptaugenmerk auf der Entwicklung von XForms Anwendungen. Lars wurde 2006 Teil des Chiba Projekts und hat Anfang 2009 mit Joern betterFORM gegründet.

Lars entwirft und setzt XForms basierte Anwendungen um. Hierbei kommen auch weitere XML Kerntechnologien wie z.B. XSLT, XSD abder auch Java, JavaScript und CSS zum Einsatz. Er betreut Kunden durch den gesamten Software-Lebenszyklus. Angefangen beim Aufnehmen der Anforderungen bis zur Auslieferung der fertigen Software.

Tobi Krebs ist ein Software Architekt und XForms Experte der mit Hilfe von XML-Technologien (wie XForms, XSLT, XQuery und XML-basierte Datenbanken), Java and JavaScript weltweit Software Lösungen für Kunden entwirft und implementiert. Er stieß Mitte 2009 zu Jörn und Lars.

Zuvor arbeitete er als Java Entwickler für die "Gesellschaft für Informationssysteme und Prozessautomation mbH".

Fabian Otto ist seit seiner Kindheit Computernutzer und Programmierer. Er hat eine Vorliebe für merkwürdige und dynamische Programiersprachen und nutzt ausschließlich *NIX - ähnliche Betriebssysteme. Er arbeitet seit Herbst 2009 im betterFORM Team.
