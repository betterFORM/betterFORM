The upload Element
Description

This form control enables the common feature found on Web sites to upload a file from the local file system, as well as accepting input from various devices including microphones, pens, and digital cameras.

XForms 1.1 Links
Recommendation
Quick Reference
File Upload Control

An upload control binding to datatype 'anyURI' is equivalent to a file upload known from HTML.

XForms Markup

<!--XForms UI-->
<xf:upload id="upload1" ref="items/item[1]/value">
    <xf:label>a standard file upload</xf:label>
    <xf:filename ref="attribute::file"></xf:filename>
    <xf:mediatype ref="attribute::type"></xf:mediatype>
</xf:upload><xf:group appearance="bf:verticalTable">
    <xf:output ref="items/item[1]/value/@file">
        <xf:label>File:</xf:label>
    </xf:output>
    <xf:output ref="items/item[1]/value/@type">
        <xf:label>Type:</xf:label>
    </xf:output>
    <xf:output value="items/item[1]/value">
        <xf:label>Download Link:</xf:label>
    </xf:output>
</xf:group>

a standard file upload File: Type: Download Link:

Upload as base64Binary data

An upload control binding to datatype 'base64Binary' encodes the uploaded data as base64Binary and stores these as the value of the bound node.
